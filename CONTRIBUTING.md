# Contributing to textract-py3

This is a maintained fork of [deanmalmgren/textract](https://github.com/deanmalmgren/textract).

For contribution guidelines, see the [contributing documentation](https://textract.readthedocs.io/en/latest/contributing.html).

## Quick Start

```bash
git clone https://github.com/KyleKing/textract-py3.git
cd textract-py3
uv sync --group dev
uv run pytest
```
